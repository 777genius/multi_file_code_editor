# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-12

### Added
- Initial release of TypeScript Language Support plugin
- 50 comprehensive TypeScript code snippets with type annotations
- Support for .ts, .tsx file extensions
- Syntax highlighting via Monaco Editor
- Bracket matching and autocomplete
- Functions with full type signatures
- Interfaces, types, generics snippets
- Class snippets with access modifiers
- Decorator snippets (class, method, property)
- Async/await with Promise types
- ES6+ module imports/exports with type-only variants
- Utility types (Partial, Readonly, Record, Pick, Omit)
- Console and debugging snippets
